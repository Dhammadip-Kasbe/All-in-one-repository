# 🧮 All-In-One Calculator

A feature-rich, multi-mode calculator built with pure HTML, CSS, and JavaScript.

## Modes
- **Basic** — Arithmetic + memory bank (MC/MR/M+/MS)
- **Scientific** — 30+ functions: trig, logs, powers, factorial, constants
- **Programmer** — Bitwise ops, base conversion (DEC/HEX/OCT/BIN)
- **Finance** — EMI, compound interest, tip splitter, discount calculator
- **Statistics** — Mean, median, σ, variance, min/max from raw data

## Features
- Retro terminal aesthetic with CRT scanline display
- Ripple click animations & keyboard support
- Light / Dark theme toggle
- Zero dependencies — single HTML + CSS + JS
